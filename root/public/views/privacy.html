Mates is an end to end encrypted real time group chat app.


What does this mean?


It means our aim is to store as little information about you as possible. End to end means only the sender and receiver have access to the message.


If your message is sent long distance certain metadata can be seen about the message on every server the message passes through


What happens when I send a message?

Once you hit send, the message is encrypted and sent to the receiver. The receiver decrypts the message automatically.

What information do we keep?

Only your username and password

So if our database is hacked this information could potentially be leaked

What about messages?

They are end to end encrypted, so only someone with access to your phone can read them
